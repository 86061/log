Write Static HTML Generator Using Netlify Function into Github

Author: Master

Body:

Tidak secara otomatis. Form Netlify akan mengirimkan data ke endpoint default Netlify atau endpoint lain yang Anda tentukan. Agar form submission dapat memicu fungsi saveForm.js, Anda perlu mengkonfigurasinya agar form tersebut diarahkan ke endpoint dari fungsi tersebut.

Berikut langkah-langkah memastikan bahwa submit form Netlify memicu fungsi saveForm.js:

1. Konfigurasi Form Netlify untuk Trigger Fungsi

Form Netlify secara default mengirim data ke backend Netlify untuk diproses. Untuk mengarahkan form submission ke fungsi saveForm.js:

  • Pastikan file saveForm.js terletak di folder /netlify/functions.
  • Tambahkan atribut action dan method ke form:
<form name="postForm" action="/.netlify/functions/saveForm" method="POST" netlify netlify-honeypot="bot-field">
  <input type="text" name="title" placeholder="Title" required />
  <input type="text" name="slug" placeholder="Slug" required />
  <input type="text" name="tags" placeholder="Tags" />
  <input type="text" name="category" placeholder="Category" />
  <textarea name="bodypost" placeholder="Post Content"></textarea>
  <input type="text" name="author" placeholder="Author" />
  <input type="file" name="imagefile" />

  <button type="submit">Submit</button>
</form>

Penjelasan:

  • action="/.netlify/functions/saveForm": Mengarahkan submission ke fungsi saveForm.js.
  • method="POST": Menggunakan metode HTTP POST untuk mengirim data.

2. Deploy Fungsi ke Netlify

Netlify akan secara otomatis mem-build dan men-deploy file di /netlify/functions. Pastikan:

  1. Anda sudah mengaktifkan fungsi serverless Netlify.
  2. Setelah deploy, pastikan fungsi dapat diakses di: 
    https://<your-site>.netlify.app/.netlify/functions/saveForm

3. Testing

Setelah deployment:

  1. Submit form di halaman Anda.
  2. Data dari form akan dikirim ke endpoint /saveForm.
  3. Fungsi saveForm.js akan dijalankan dan memproses data form untuk disimpan ke GitHub.

Jika semuanya benar, setiap kali form disubmit, fungsi saveForm.js akan otomatis dipicu.
